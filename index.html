폴더 구조

필요한 폴더들

프로젝트/ ├── index.html (이 파일) ├── admin/ │ ├── index.html (CMS 관리자) │ └── config.yml (CMS 설정) ├── _data/ │ └── settings.yml (사이트 설정) ├── _posts/ │ ├── news/ (뉴스 폴더) │ └── resources/ (자료 폴더) └── assets/ └── uploads/ (업로드 폴더)
1

GitHub 저장소 준비

  • • GitHub에서 새 저장소 생성 (예: osandae-church)
  • • 위 파일들을 모두 저장소에 업로드
  • • 폴더 구조가 정확한지 확인
2

Netlify 사이트 연결

  • • netlify.com에서 회원가입
  • • "New site from Git" 선택
  • • GitHub 저장소 연결
  • • Deploy site 클릭
3

Netlify Identity 설정

  • • Site settings → Identity 메뉴
  • • "Enable Identity" 클릭
  • • Registration: "Invite only" 선택
  • • Services → "Enable Git Gateway" 클릭
4

관리자 계정 생성

  • • Identity → Users → "Invite users"
  • • 관리자 이메일 입력 후 초대
  • • 이메일 확인하여 비밀번호 설정
  • • YOUR_SITE.netlify.app/admin/ 에서 로그인

뉴스 관리

뉴스 작성 방법:
  • 1. CMS 관리자 페이지 접속
  • 2. "교회 소식" 메뉴 클릭
  • 3. "New 교회 소식" 버튼 클릭
  • 4. 제목, 내용, 카테고리 입력
  • 5. 이미지 첨부 (선택사항)
  • 6. "Publish" → "Save" 클릭
자동 기능:
